## Task 1: 启动后端服务

在 `service/` 目录下启动 FastAPI 后端服务：


## 风险与注意事项

1. **数据库状态**: 需确保 `dev.db` 中有种子数据（admin 用户、基础角色、菜单数据），否则页面会显示空数据
2. **端口冲突**: 确保 8000 和 8848 端口未被占用
3. **Redis**: 后端配置了 Redis，开发环境下如果 Redis 未启动，需确认缓存模块是否有降级处理
4. **CORS**: 前端 8848 已在后端 CORS 白名单中，跨域应无问题
5. **删除操作**: 测试删除时注意不要删除 admin 用户和核心角色/菜单，避免影响后续测试
