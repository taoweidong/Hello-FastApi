    component "entities/ 领域实体" as DOM_ENT

    component "services/ 领域服务" as DOM_SVC

    component "cache/ Redis" as INF_CACHE
    component "http/ 中间件 + 异常处理" as INF_HTTP

API_V1 -down-> API_DEP : 调用
API_V1 -up-> API_RES : 返回
API_DEP -down-> APP_SVC : 创建

APP_SVC -down-> DOM_REPO : 调用
APP_SVC -down-> DOM_SVC : 调用
APP_SVC ..> INF_CACHE : ⚠️ 直接导入(待优化)

DOM_REPO -up-> APP_SVC : 返回领域实体
INF_REPO -up-> DOM_REPO : 实现
INF_REPO -down-> INF_DB : 使用
INF_REPO .> DOM_ENT : to/from_domain

legend
  🟢 绿色标记 = 已激活/已集成的组件
  🟡 黄色标记 = 存在分层约束违规的组件

## 4. 仓储层架构图（重构后）

```plantuml
@startuml
!theme plain

skinparam componentStyle uml2

class GenericRepository {
    +get_by_id(id): T | None
    +get_all(): list[T]
    +get_one_by(field, value): T | None
    +exists(field, value): bool
    +create(entity): T
    +update(entity): T
    +delete(id): bool
    +batch_delete(ids): int
    +_apply_filters(stmt, filters): Select
}

interface UserRepositoryInterface {
    +get_by_username(username): UserEntity | None
    +get_by_email(email): UserEntity | None
    +get_user_roles(user_id): list[RoleEntity]
    +get_user_all_menus(user_id): list[MenuEntity]
}

interface MenuRepositoryInterface {
    +get_user_menus(user_id): list[MenuEntity]
    +get_role_menu_ids(role_ids): list[str]
}

class UserRepository {
    +get_by_username(username)
    +get_by_email(email)
    +get_user_roles(user_id)
    +get_user_all_menus(user_id)
    +update_status(user_id, status)
    +reset_password(user_id, new_password)
}

class MenuRepository {
    +get_user_menus(user_id)
    +get_role_menu_ids(role_ids)
    +delete_with_children(menu_id)
}

GenericRepository <|-- UserRepository
GenericRepository <|-- MenuRepository

note right of GenericRepository
  基于 SQLModel Native API
  使用 session.exec() 查询
  移除 fastcrud 依赖
end note

@enduml
```

## 6. 关键技术栈

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| Web 框架 | FastAPI 0.115+ | ASGI + Pydantic v2 |
| ORM | SQLModel 0.0.22+ | SQLAlchemy 2.0 + Pydantic |
| 仓储层 | GenericRepository (自研) | SQLModel Native API，移除 fastcrud |
| 数据库 | PostgreSQL (生产) / SQLite (测试) | AsyncDriver |
| 缓存 | Redis 5.0+ | 含降级策略 |
| 认证 | JWT (python-jose) + bcrypt | Token + 密码哈希 |
| API 风格 | classy-fastapi | 类装饰器路由 |
| 限流 | slowapi + limits | 固定窗口限流 |
| 日志 | loguru | 结构化日志 |
| 文档 | Swagger UI + ReDoc | /api/docs |

> 🟢 已移除 `fastcrud` 依赖，改用基于 SQLModel Native API 的 `GenericRepository`
